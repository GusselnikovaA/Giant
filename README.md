# Giant
Landing page Giant

###  Чтобы развернуть проект необходимо:
```sh
$ git clone https://github.com/GusselnikovaA/Giant.git
$ npm i 
then open index.html
```

###  Скрипты package.json:

| Скрипт | Назначение |
| ------ | ------ |
| gulp build | Запустить development-режим сборки. |


### Stack
+ jQuery
+ Slick slider
+ SASS
+ Gulp 4.0.2
+ FontAwesome
